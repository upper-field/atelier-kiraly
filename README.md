# BUILD

```
$ cd free-game # or free-music or music
$ bundle install
$ bundle exec jekyll build
$ open _site
```

# WRITE HTML

```
$ cd free-game # or free-music or music
$ bundle exec jekyll s
// You should access http://localhost:4000/ with your browser.
```
